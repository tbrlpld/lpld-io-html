CAE Job Diary

Description

CAE Job Dairy is a web application to search, track and summarize simulation jobs.

I developed CAE Job Dairy as a side project to help me stay organized in my day job as a CAE simulation and development engineer at Joyson Safety System's advance development department for occupant restraint systems. Especially, when many simulation jobs had to be handled it helped me to see which jobs are running, pending or finished. Which jobs still require an analysis? And what was the outcome of that job again?

CAE Job Diary is developed as a Django web application. The web application allows the user to search, view and edit the jobs. The creation of jobs is handled by a back end process that polls log files of the queueing system. A second back end process checks the status of the jobs by parsing the file structure in the corresponding directories.

The source code hosted on Gitlab as a private repository. The repository is still private because some example files and installation/deployment instructions still contain sensitive information. I am currently working on removing the sensitive information and will make the repository publicly viewable once that is done.

RememberMe

Description

RememberMe is a JavaScript memory game that I developed as the final project of the Udacity Nanodegree "Intro to Programming" in 2018.

The Nanodegree program seems to have been restructured since. The projects are different now then they used to be, but the overall syllabus still seems similar with HTML, CSS, JavaScript, Python and Object-Oriented Programming.

The topic of the final project was predefined and the user side requirements where defined (memory game logic, timer, move counter, star rating, congratulations, etc.). A design mock-up was provided, but not strictly requested for implementation. It was my task to turn the requirements into a working JavaScript browser game.

The source code is publicly available on my GitHub profile and a live version can be reached by the link below or the image above.
